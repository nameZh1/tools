# Architect Mode Rules

## 非显而易见的架构约束

- **无状态设计**: 服务器无会话/认证，所有数据直接存文件系统
- **单进程限制**: 无集群/负载均衡支持，仅适合小规模局域网使用
- **文件存储耦合**: 上传文件直接存 `shared_files/`，文件名即标识符（无数据库映射）
- **前后端同源**: 前端由 Express 静态服务，无 CORS 问题但也无分离部署能力
- **同步文件操作**: 部分文件操作使用 `fs.statSync`/`fs.existsSync`（[`server.js:40,66,100`](server.js:40)），高并发可能阻塞
